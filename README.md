## summary

| Command                 | Description                     |
| ----------------------- | ------------------------------- |
| `psql -U postgres`      | Start psql with user `postgres` |
| `\l`                    | List databases                  |
| `CREATE DATABASE mydb;` | Create a new database           |
| `\c mydb`               | Connect to a database           |
| `\q`                    | Quit `psql`                     |
